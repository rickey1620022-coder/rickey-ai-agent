# 🤖 Rickey AI Agent

A smart personal AI business assistant — installable as a mobile app (PWA).

## 🌐 Live App
👉 **https://rickey1620022.github.io/rickey-ai-agent**

## ✨ Features
- ✅ **AI To-Do** — speak or type tasks, AI auto-categorises
- 📞 **Call Reminders** — payment & sales reminders with date/time
- 📒 **Smart Notes DB** — AI auto-titles, tags and saves notes
- 📷 **Photo Gallery** — tag photos, search by tag, camera support
- 💬 **WhatsApp** — AI writes messages, contact book, schedule send
- 🔗 **Google Integration** — Contacts, Calendar, Gmail
- 🗄️ **Memory DB** — backup/restore all data

## 📱 Install as Mobile App
1. Open the live URL in Chrome (Android) or Safari (iPhone)
2. Tap **⋮ menu → Add to Home Screen** (Android)
3. Tap **Share → Add to Home Screen** (iPhone)

## 🔧 Tech
- Pure HTML/CSS/JS — single file, no framework
- Claude AI (Anthropic) for all AI features
- Google OAuth for Contacts, Calendar, Gmail
- localStorage for offline data storage
- PWA with service worker for offline support
