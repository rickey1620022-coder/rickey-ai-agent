🔧 Step 1 — One-Time Setup (5 min)
Google requires a free Client ID to allow this app to access your account. Do this once:
1
Go to console.cloud.google.com → Sign in with your Google account
2
Click Select Project → New Project → Name it "Rickey AI" → Create
3
APIs & Services → Enable APIs → Enable: Google People API, Google Calendar API, Tasks API, Gmail API, Google Drive API
4
APIs & Services → OAuth consent screen → External → Fill app name "Rickey AI" → Add your email → Save
5
Credentials → + Create Credentials → OAuth Client ID → Type: Web application
6
Under Authorised JavaScript origins add: loading...
7
Click Create → Copy your Client ID → Paste below
Your Client ID is saved locally on this device only. Nothing is sent to any server.
